POWER ACTIVATED!

You built the JavaScript toggle! classList.toggle() is like a super-switch — it adds the class if missing, removes it if present. One line does both!

document.body.classList.toggle("dark-mode");
// ↑ Adds OR removes "dark-mode" class automatically
🎨

COLOURS UNLOCKED!

CSS Variables are like magic paint cans! Define them once in :root, change them in .dark-mode, and everything updates automatically.

:root { --bg-color: #ffffff; }
body.dark-mode { --bg-color: #121212; }
// ↑ One change updates the whole page!
🏆
