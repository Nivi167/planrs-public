🁢 The Domino Chain — Understanding Induction

Mathematical induction is like toppling dominoes! If the 1st domino falls (Base Case), and whenever domino k falls, domino k+1 also falls (Inductive Step) — then ALL dominoes fall forever! Click the first domino to start the chain!

Claim: 1 + 2 + 3 + … + n = n(n+1)/2 for all n ≥ 1
👆 Click domino n=1 to begin the Base Case!
📖 The Two Magical Steps
  • Base Case (n=1): Verify the formula works for n=1. We check: 1 = 1(1+1)/2 = 1 ✓
  • Inductive Hypothesis: Assume it's true for some k ≥ 1. So assume: 1+2+…+k = k(k+1)/2
  • Inductive Step: Prove it for k+1. Add (k+1) to both sides: 1+2+…+k+(k+1) = k(k+1)/2 + (k+1)
  • Simplify: = (k+1)[k/2 + 1] = (k+1)(k+2)/2 = (k+1)((k+1)+1)/2 ✓
  • Conclusion: By induction, 1+2+…+n = n(n+1)/2 for ALL n ≥ 1! 🎉
