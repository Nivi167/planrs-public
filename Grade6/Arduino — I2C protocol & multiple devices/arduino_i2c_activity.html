Activity 3 — I2C Communication Chain Reaction
Arrange the steps of an I2C read transaction in the correct order by dragging them into the sequence slots.
Hint: Every I2C transaction follows: START → Address → R/W bit → ACK → Data → STOP

📦 Slave transmits data bytes
🟢 Master sends START condition
✅ Master sends ACK / NACK
